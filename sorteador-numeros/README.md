# sorteio do numero secreto

## Descrição do projeto
projeto de pratica desenvolvido na trilha de logica de programação na alura.
a parte do frontend veio pronta, modificações feitas no js.

## funcinalidades

ler a quantidades de numeros que seram sorterados e definir entre quais valores ele deve esta e exibir para o usuario

## tecnologias 

- javaScript
- HTML, CSS
- Jest (testes)

## Como rodar os testes

Para visualizar todos os testes funcionais:

```bash
# 1. Instale as dependências
npm install

# 2. Execute os testes
npm test
```

O comando `npm test` vai executar **7 testes** que validam:
- ✅ Geração de números aleatórios
- ✅ Sorteio com entrada válida
- ✅ Validação de campos vazios
- ✅ Validação de números negativos
- ✅ Validação de intervalo (de > até)
- ✅ Função reiniciar
- ✅ Troca de status do botão
